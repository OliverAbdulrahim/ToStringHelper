ToStringHelper

A project loosely based on the MoreObjects.ToStringHelper class featured in Google's Guava library.

Featured Classes

There are two variants of "to string" helper classes featured in this repository as delineated below:

Name Description
ToStringHelper A customizable String representation of an arbitrary object.
AutoToStringHelper An automatic String representation of an given object.

See the wiki for additional a walkthrough of these classes!
